1 Install Pending

▼

Install the framework with a single command. The installer sets up Python, dependencies, the environment file, and optionally starts local infrastructure.

# Linux / macOS
curl -fsSL https://raw.githubusercontent.com/pmkrafts/agile_ai_framework/main/scripts/install.sh | bash
# Windows (PowerShell)
iex (irm https://raw.githubusercontent.com/pmkrafts/agile_ai_framework/main/scripts/install.ps1)
Prefer a local install? Clone the repo and run ./scripts/install.sh (Linux/macOS) or .\scripts\install.ps1 (Windows).

5 Run a Sprint Simulation Pending

▼

Run a full AI-native sprint simulation without real LLM costs:

python -m src.simulations.sprint_simulation

Expected output includes:

  • PRD generation
  • Sprint planning
  • Architecture design
  • Coder/Test/Review cycles
  • Deployment preparation
  • KPI dashboard and cost summary
Success criteria: Simulation completes with final phase = "complete" and no errors.

6 Run the Self-Learning Loop Pending

▼

Run a sprint simulation followed by the self-learning loop:

python -m src.simulations.sprint_simulation --learning

The loop performs these stages:

  1. Observe: Collect feedback from sprint execution
  2. Analyze: Identify recurring failure patterns and KPI gaps
  3. Propose: Generate improvements to prompts/skills/governance
  4. Approve: CTO approval workflow
  5. Apply: Safely apply changes with backups
  6. Validate: Re-run simulation and compare KPIs
  7. Rollback: Revert changes that cause regression
# Run self-learning loop directly
python -m src.learning.self_learning_loop
